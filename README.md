# CC-A1
AAU Klagenfurt - Clean Code - Assignment 1

## Pre-Requirements
* Java 17
* git
* OS: Windows, macOS, Linux or compatible

## Build
`./mvnw clean package`

## Run
`java [-DapiKey="<apiKey>"] -jar target/webcrawler.jar [--url "<url>"] [--depth <depth>] [--lang <language>]`

## Test
`./mvnw test`
